Objetos

Un objeto será como una caja en la que vamos a poder almacenar otras cajas. Es decir, tendremos una variable y dentro de esta variable se podrán almacenar más variables o funciones. Por ejemplo, se ha estado usando la función "document.write()". Lo que realmente se ha estado haciendo es del objeto document, usar la variable write que almacena una función. El operador "." sirve para obtener los contenidos de un objeto.

Definiendo objetos a partir de funciones.

Hay varias formas de definir un objeto en Javascript. Lo más común es usar una función. La palabra clave this se referirá al objeto que estamos creando o manejando. Como se puede ver dentro de la función Alumno se hacen varias referencias a this que es el objeto que se está creando. A las variables qeu se definen dentro de un ojeto se las llama atributos. A las funciones que se definen dentro de un objeto se les llama métodos.
