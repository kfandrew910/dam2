Variables
Como hemos estudiado en primero, una variable es un espacio de memoria RAM al que le asignamos un nombre y en el que guardamos un valor que puede ser alterado a lo largo de la ejecución de nuestro programa.
Estas variables pueden ser inicializadas en la declaración o posteriormente.
Para declarar una variable, utilizamos la palabra var
El Operador de asignación.

Para asignar un valor a una variable al inicializarla o con posteriordad, debemos utilizar el operador de asignación = como podemos observar en la imagen anterior.

Además del operador =, también existen += y -=

Tipo Booleano.

Este tipo de dato únicamente puede tomar dos tipos de valores:

  • Verdadero
  • Falso

Tipo Numérico.

Este tipo de dato puede contener valores numéricos reales y enteros. Las operaciones que podemos hacer sobre ellos son:

  • Suma +
  • Resta -
  • Multiplicación *
  • División /
  • En valores enteros (sin decimales) nos encontramos tres operaciones más:
    • Resto de una división %
    • Aumentar en uno el valor del número ++
    • Disminuir en uno el valor del número --
Los operadores ++ y -- actúan de manera diferente cuando se colocan antes o después de la variable.
  • Cuando las ponemos delante de la variable incrementa/decrementa el valor de la variable antes de evaluar ninguna otra variable.
  • Cuando las ponemos detrás de la variable se incrementa/decrementa el valor de la variable después de haber superado el ;

Tipo Texto.

Este tipo de variable va a guardar texto.

  • Para guardar una cadena de texto podemos utilizar las comillas dobles " o las comillas simples '
  • Con las comillas dobles o simples únicamente podemos insertar una línea de texto, si queremos insertar más de una línea, utilizamos el acento invertido `.
    Ojo!!!, eso no quiere decir que te vayan a aparecer varias líneas en tu página web.
  • La única operación que podemos realizar con las cadenas de texto es la concatenación que la realizamos a través del operador +.

Conversión Implícita de Tipos

Cuando trabajamos con variables de distinto tipo, debemos entender que JavaScript funcionará de una forma o de otra atendiendo a las siguientes situaciones:

  • Si tenemos una variable de tipo texto y otra numérica y las sumamos, JavaScript convierte ese número a cadena de texto y lo concatena a la variable de texto.
  • Si son dos cadenas de texto que representan un número e intentamos concatenarlas con el operador *, lo que nos devolverá es la multiplicación de esos dos valores, aunque sean cadenas de texto. Esto es una interpretación que hace JavaScript
  • Si es una cadena de texto que no es un número y una cadena de texto que representa un número e intentamos utilizar el operador *, nos devolverá el código de error NaN (Not a Number) indicando que no se puede realizar la operación.

Conversión Explícita de Tipos

En determinadas situaciones nos puede interesar hacer una conversión de un tipo de texto a un numérico de forma explicita. JavaScript nos ofrece dos funciones para poder realizarlo.

  • parseInt(string): Convierte una cadena de texto en un número entero, es decir, sin decimales.
  • parseFloat(string): Convierte una cadena de texto en un número real, es decir, con decimales.

Undefined y Null

Hay dos conceptos sobre las variables que es importante que tengamos en cuenta:

  • Undifined: La variable está declarada pero no se le ha asignado ningún valor.
  • Null: Cuando asignamos este valor a una variable, lo que hacemos es borrar ese espacio reservado de la memoria RAM, de tal forma que no dejamos basura que nos puede dar problemas mas adelante.

Ventanas Modales.

JavaScript nos proporciona ventanas modales para visualizar y/o recoger datos.

  • alert(mensaje): Le muestra al usuario un mensaje.
  • prompt(mensaje): Le pedimos un dato al usuario con el que haremos alguna operación. Este dato siempre se va a guardar como un tipo string. Como observamos en la imagen, el prompt siempre se le asigna a una variable que es la que va a guardar el valor introducido por el usuario.
